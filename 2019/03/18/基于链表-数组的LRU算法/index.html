基于链表/数组的LRU算法

单向链表

  1. 每当有新数据被访问时，首先遍历链表，处在链表尾结点的数据是需要淘汰的数据
  2. 若数据存在链表中，从原位置删除，插入到链表头部
  3. 若数据不存在链表中，链表空间足够，则将该数据插入头部
  4. 若数据不存在链表中，链表空间不足，则将尾结点删除，并插入数据到头部

时间复杂度为$O(n)$

数组

  1. 每当有新数据被访问时，首先遍历数组，处在数组首位的数据是需要淘汰的数据
  2. 若数据存在数组中，则将该数据从当前位置删除，添加至数组最后一位
  3. 若数据不存在数组中，且数组空间足够，将数据添加至最后一位
  4. 若数据不存在数组中，且数组空间不足，则将数组首位的数据删除，并添加至数组最后一位

时间复杂度为$O(n)$
